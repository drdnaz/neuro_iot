# NEURO-SENTINEL: Hibrit Yapay Zeka Adli Bilişim ve IoT Proje Durumu

Bu doküman, endüstriyel elektrik panosu güvenlik sistemi olan **Neuro-Sentinel** projesinin en güncel mimarisini, donanım pin haritasını ve çalışan kod tabanını içerir. Antigravity asistanı için geliştirme bağlamı (context) sağlamak amacıyla hazırlanmıştır.

---

## 1. Sistem Mimarisi (Hibrit Yapay Zeka)
Proje, endüstriyel tesislerde sıfır güven (zero-trust) güvenliği ve internet bağlantısı koptuğunda bile kesintisiz çalışmayı sağlamak için **Hibrit Yapay Zeka (Edge + Yerel LLM) Mimarisi** kullanmaktadır.

1. **Uç Nokta (ESP32-S3):** Sistemin omuriliğidir. 4 farklı protokol (1-Wire, Analog ADC, I2S Rx ve Dijital GPIO) kullanarak 4 sensörü (Sıcaklık, Piroliz Gazı, Akustik Ark ve Hareket) sürekli tarar. Kritik eşikler aşıldığında fiziksel sireni (MAX98357A Amfi üzerinden) çalmak gibi gerçek zamanlı donanım reflekslerini yönetir.
2. **Yerel Beyin (Ollama / Llama 3):** Sistemin adli bilişim uzmanıdır. Sensör verileri USB Seri port üzerinden yerel bilgisayara akar. Bir kriz anında, Python köprü yazılımı yerel bilgisayardaki **Ollama (Llama 3)** modelini tetikleyerek İngilizce formatta yüksek düzeyde teknik bir mühendislik raporu (Forensic Report) oluşturur.

---

## 2. Güncel Donanım Konfigürasyonu ve Pin Haritası

### Ana Sensörler ve Aktüatörler
| Bileşen / Hat | ESP32-S3 Pini | Breadboard Bağlantısı / Durum | Durum |
| :--- | :--- | :--- | :--- |
| **GND** | GND | Mavi (-) Hat | Bağlı |
| **5V Güç** | 5V / Vin | Sol Kırmızı (+) Hat (MQ-135 VCC için) | Bağlı |
| **3.3V Güç** | 3.3V | Sağ Kırmızı (+) Hat (DS18B20, INMP441, MAX98357A) | Bağlı |
| **DS18B20 (Isı)** | **GPIO 4** | VCC -> 3.3V Hattı (GPIO 4 ile VCC arasında 4.7k Pull-up direnç var) | Çalışıyor |
| **MQ-135 (Gaz)** | **GPIO 2** | ADC1_CH1 / VCC -> 5V Hattı | Çalışıyor |
| **HC-SR501 (PIR)** | **GPIO 1** | Dijital Giriş | Çalışıyor |
| **INMP441 (I2S Mikrofon)** | **SD: GPIO 5 <br> SCK: GPIO 6 <br> WS: GPIO 7** | L/R Pini -> GND'ye bağlı (Sol Kanal) | Çalışıyor |
| **MAX98357A (Amfi)** | **BCLK: GPIO 15 <br> LRC: GPIO 16 <br> DIN: GPIO 17** | Fiziksel hoparlöre bağlı | Çalışıyor (Donanım Sireni Aktif) |

### Ekran Bağlantısı (ILI9341 SPI)
*Not: Ekran şu anda beyaz ekran (WSOD) vermektedir. Güç hatları çalışıyor ancak SPI/Komut senkronizasyonu üzerinde çalışılması gerekiyor.*
* **CS:** GPIO 10
* **RST:** GPIO 8
* **DC:** GPIO 9
* **SDI (MOSI):** GPIO 11
* **SDK / SCK (CLK):** GPIO 13
* **VCC:** 5V Hattı
* **LED:** 3.3V Hattı

## 3. Cihaz Yazılımı Tabanı (`main/neuro_sentinel_main.c`)
*Geliştirme Ortamı: ESP-IDF (C Dili)*

```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include "freertos/FreeRTOS.h"
#include "freertos/task.h"
#include "driver/gpio.h"
#include "esp_adc/adc_oneshot.h"
#include "rom/ets_sys.h"
#include "driver/i2s_std.h"

#define PIR_PIN GPIO_NUM_1
#define GAS_CHANNEL ADC_CHANNEL_1 
#define DS_PIN GPIO_NUM_4

#define I2S_MIC_SD  GPIO_NUM_5
#define I2S_MIC_SCK GPIO_NUM_6
#define I2S_MIC_WS  GPIO_NUM_7

#define I2S_AMP_BCLK GPIO_NUM_15
#define I2S_AMP_LRC  GPIO_NUM_16
#define I2S_AMP_DIN  GPIO_NUM_17

i2s_chan_handle_t rx_chan; 
i2s_chan_handle_t tx_chan; 

void init_amplifier() {
    i2s_chan_config_t chan_cfg = I2S_CHANNEL_DEFAULT_CONFIG(I2S_NUM_AUTO, I2S_ROLE_MASTER);
    i2s_new_channel(&chan_cfg, &tx_chan, NULL);
    i2s_std_config_t std_cfg = {
        .clk_cfg  = I2S_STD_CLK_DEFAULT_CONFIG(16000),
        .slot_cfg = I2S_STD_PHILIPS_SLOT_DEFAULT_CONFIG(I2S_DATA_BIT_WIDTH_16BIT, I2S_SLOT_MODE_MONO),
        .gpio_cfg = { .mclk = I2S_GPIO_UNUSED, .bclk = I2S_AMP_BCLK, .ws = I2S_AMP_LRC, .dout = I2S_AMP_DIN, .din = I2S_GPIO_UNUSED }
    };
    i2s_channel_init_std_mode(tx_chan, &std_cfg);
    i2s_channel_enable(tx_chan);
}

void play_alarm_tone() {
    int16_t tone_buf[500];
    size_t bytes_written = 0;
    for (int i = 0; i < 500; i++) {
        tone_buf[i] = (int16_t)(10000 * sin(2 * M_PI * 880 * i / 16000));
    }
    for(int k = 0; k < 3; k++) {
        i2s_channel_write(tx_chan, tone_buf, sizeof(tone_buf), &bytes_written, portMAX_DELAY);
        vTaskDelay(pdMS_TO_TICKS(100));
    }
}

void init_microphone() {
    i2s_chan_config_t chan_cfg = I2S_CHANNEL_DEFAULT_CONFIG(I2S_NUM_AUTO, I2S_ROLE_MASTER);
    i2s_new_channel(&chan_cfg, NULL, &rx_chan);
    i2s_std_config_t std_cfg = {
        .clk_cfg  = I2S_STD_CLK_DEFAULT_CONFIG(16000),
        .slot_cfg = I2S_STD_PHILIPS_SLOT_DEFAULT_CONFIG(I2S_DATA_BIT_WIDTH_32BIT, I2S_SLOT_MODE_MONO),
        .gpio_cfg = { .mclk = I2S_GPIO_UNUSED, .bclk = I2S_MIC_SCK, .ws = I2S_MIC_WS, .din = I2S_MIC_SD }
    };
    i2s_channel_init_std_mode(rx_chan, &std_cfg);
    i2s_channel_enable(rx_chan);
}

int get_audio_level() {
    int32_t raw_samples[64]; size_t bytes_read = 0;
    i2s_channel_read(rx_chan, raw_samples, sizeof(raw_samples), &bytes_read, portMAX_DELAY);
    int samples_read = bytes_read / sizeof(int32_t); int64_t total_energy = 0;
    for(int i = 0; i < samples_read; i++) total_energy += abs(raw_samples[i] >> 12);
    return (int)(total_energy / samples_read);
}

int ds_reset() {
    gpio_set_direction(DS_PIN, GPIO_MODE_OUTPUT); gpio_set_level(DS_PIN, 0); ets_delay_us(480);
    gpio_set_direction(DS_PIN, GPIO_MODE_INPUT); ets_delay_us(70);
    int presence = gpio_get_level(DS_PIN); ets_delay_us(410); return presence;
}
void ds_write_bit(int b) {
    gpio_set_direction(DS_PIN, GPIO_MODE_OUTPUT); gpio_set_level(DS_PIN, 0); ets_delay_us(b ? 5 : 60);
    gpio_set_direction(DS_PIN, GPIO_MODE_INPUT); ets_delay_us(b ? 60 : 5);
}
int ds_read_bit() {
    gpio_set_direction(DS_PIN, GPIO_MODE_OUTPUT); gpio_set_level(DS_PIN, 0); ets_delay_us(2);
    gpio_set_direction(DS_PIN, GPIO_MODE_INPUT); ets_delay_us(10);
    int b = gpio_get_level(DS_PIN); ets_delay_us(50); return b;
}
void ds_write_byte(int val) {
    for (int i = 0; i < 8; i++) { ds_write_bit(val & 0x01); val >>= 1; }
}
int ds_read_byte() {
    int val = 0; for (int i = 0; i < 8; i++) { if (ds_read_bit()) val |= (1 << i); } return val;
}
float read_temperature() {
    if (ds_reset() == 0) {
        ds_write_byte(0xCC); ds_write_byte(0x44); vTaskDelay(pdMS_TO_TICKS(750)); 
        ds_reset(); ds_write_byte(0xCC); ds_write_byte(0xBE);
        int lsb = ds_read_byte(); int msb = ds_read_byte();
        return ((msb << 8) | lsb) / 16.0;
    }
    return -99.0; 
}

void app_main(void) {
    printf("NEURO-SENTINEL Baslatiliyor...\\n");
    gpio_reset_pin(PIR_PIN); gpio_set_direction(PIR_PIN, GPIO_MODE_INPUT);
    
    adc_oneshot_unit_handle_t adc1_handle;
    adc_oneshot_unit_init_cfg_t init_config1 = { .unit_id = ADC_UNIT_1 };
    adc_oneshot_new_unit(&init_config1, &adc1_handle);
    adc_oneshot_chan_cfg_t config = { .bitwidth = ADC_BITWIDTH_DEFAULT, .atten = ADC_ATTEN_DB_12 };
    adc_oneshot_config_channel(adc1_handle, GAS_CHANNEL, &config);
    
    init_microphone();
    init_amplifier();

    while (1) {
        int pir_val = gpio_get_level(PIR_PIN);
        int gas_raw = 0; adc_oneshot_read(adc1_handle, GAS_CHANNEL, &gas_raw);
        int audio_lvl = get_audio_level();
        float temp = read_temperature(); 

        if(temp != -99.0) {
            printf("[VERİ] ISI: %.2f C | GAZ: %4d | SES (ARK): %4d | HAREKET: %d \\n", temp, gas_raw, audio_lvl, pir_val);
            if (gas_raw > 1500 || temp > 35 || audio_lvl > 3000) {
                play_alarm_tone();
            }
        }
    }
